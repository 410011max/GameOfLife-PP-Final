# GameOfLife
A fast, C++ OpenGL implementation of Conway's Game of Life with 2D & 3D version!

## Keyboard Commands:
  [1...9] : Simulation Size
  
  [Tab] : toggle 2D/3D
  
  [Space] : Start/Stop Simulation
  
  [+],[-] : Zoom in/out
  
  [w],[s],[a],[d] : Pan
  
  [r] : toggle rotation
  
  [c] : toggle shading style
